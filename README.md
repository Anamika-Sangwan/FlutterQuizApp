# FlutterQuizApp
An Online Quiz App with Result Analysis

Features
1) Quiz Interface:
    Progress indicator showing current question
    Category badges for each question
    Clean, modern UI with Material Design 3
2) Tracks time spent on each question
    Result Analysis:
    Overall score with percentage
    Performance metrics (total time, average time, accuracy)
    Category-wise breakdown with visual progress bars
      Detailed question review showing:
      Correct/incorrect answers
      Your answer vs correct answer
    Time spent per question
    Color-coded results (green for correct, red for incorrect)
3) Additional Features:
      Restart quiz functionality
      Motivational messages based on performance
      Responsive design
      Trophy icon for high scores
